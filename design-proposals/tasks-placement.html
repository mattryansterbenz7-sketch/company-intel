Why option 1 wins

  • First-class placement without eating space. Gets its own horizontal band — not squeezed into the header, not a sidebar stealing stat-row width. 54px of vertical.
  • Inline edit + done + menu — task card is big enough to land on, checkbox is on-target, ⋯ menu reveals on hover (snooze / reschedule / delete).
  • Filter chips have a home — top-right of the band. Today / This week / By company / Done.
  • Horizontal scroll — doesn't grow the dashboard vertically if you have 10 tasks. Scale is handled.
  • "+ Add task" terminal card — task creation where task management lives. Inline composer (title field + company picker + date) when clicked.
  • Done state strike-through + fade — honest state cue, matches the existing behavior already in the header.
