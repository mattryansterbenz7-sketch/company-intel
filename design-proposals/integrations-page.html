Section 7

Open questions

Three calls that shape the build.

  1. OpenAI — Required or Recommended? Chat functions without it when Claude is up, but the fallback chain degrades. Current mockup shows Recommended. Upgrade to Required if we treat fallback as non-optional.
  2. First-time-user guidance — inline or dedicated flow? The tiered page itself tells the story. Option A: lean on the page (this mockup). Option B: wrap a first-run setup modal that hands users off to this page once Anthropic is in.
  3. Cost hints — static copy or pulled from real usage? Current design shows hardcoded ranges ("<$5/mo"). We have per-model cost tracking in api.js — could surface the user's actual 30-day spend instead, which is more honest but more complex.
