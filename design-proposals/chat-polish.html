  • Kill: italic "Hey, I'm Coop" greeting; uppercase tiny-caps on markdown h3 (company.js:5146); uppercase+letter-spacing on model button (chat.js:156); collapsible gray box for tool-use (ui-utils.js:161+); bouncing-dots "thinking" animation (company.html:851-855); all emoji/symbol-as-icon on model list.

  • Tool-use as first-class inline: replace renderContextManifest's collapsible box. Each tool call is a one-line inline row inside the forming assistant message: colored dot (by category) + "Read {thing} from {source}" or "Pulled {N} {kind}" with a right-aligned state (in progress → ✓). Use a subtle amber progress-line for active calls (no spinner).
  • Model indicator: sentence case ("Sonnet 4.6" / "Haiku 4.5" / "GPT-4.1"), small colored dot per provider (green=OpenAI, coral=Anthropic, amber=Haiku), compact chip "Using [model]" above the body. Click chip cycles to next available model. Fallback note is natural language: · fell back from Sonnet.
  • Header: left block = avatar + "Coop" title + surface subtitle ("Krossings · Head of GTM"). Right block = 3 controls (maximize, minimize, close), visually separated. No mixing of identity and controls.
  • Empty state: sentence-case greeting that asks a question ("What should we figure out?") rather than a marketing pitch. Prompts are contextual to the surface.
  • Message rendering: user message gets a dark pill; assistant messages are plain text with real typographic hierarchy (h4, ul, p, strong) — no bubble, no gradient, no avatar glow. Line-height 1.65 on body, 10px paragraph spacing.
  • Motion: panel open fades + scales 0.96→1 over 200ms ease-out. Panel close reverse, 120ms ease-in. Message-added fades in 120ms.
